Производство серной кислоты

Для начала перенесём производство электричества над базой, чтобы для производства оно нам не мешало.

Для производства серной кислоты нам потребуются железные плиты, сера и вода.

Химические заводы расположим выше производства серы, чтобы нефтепродукты перерабатывались все в одном месте. Серу и железные плиты выведем на один конвейер и с помощью длинных манипуляторов будем подавать в заводы. Воду пустим по трубам. Готовая серная кислота будет храниться в резервуарах.

Сейчас у нас начинаются проблемы с нехваткой нефти и других ресурсов с ней связанных. Поэтому, поставим ещё несколько нефтеперерабатывающих заводов.

Так же у нас есть возможность поставить ещё несколько нефтяных вышек на добычу самой нефти, поэтому самое время это сделать.

Эти улучшения по нефти позволили стабилизировать её расходы хотя-бы на некоторое время.
